Stuck at work today? Let’s have fun with powershell. Open on your powershell and create a quick text-to-speech funtion: function say-this { param ( [Parameter(Mandatory=$true)] $Text ) $object = New-Object -ComObject SAPI.SpVoice $object.Speak($text) | Out-Null } Now you have a function called ‘say-this’ you can name it what ever you want.
